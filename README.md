# Car_Page
Este proyecto debe ser lo más parecido posible a la muestra del vídeo, incluyendo sus funcionalidades.


# Desarrollo de la página web
Debéis crear la página web siguiendo el vídeo adjunto. La página debe ser lo más parecida posible a la que se muestra en el vídeo y debe incluir las funcionalidades que aparecen en él.

# Estructura del proyecto
El proyecto debe estar organizado en los siguientes archivos:

basededades.js: Gestionará la “base de datos”, incluyendo las funciones para cargar, insertar y eliminar elementos. Estos elementos deben ser objetos con herencia, como hicimos en la última práctica de vehículos.

scripts.js: Contendrá todas las funcionalidades necesarias para manipular el DOM y mostrar los datos de manera adecuada.

index.html: La página principal del proyecto, donde se mostrará toda la información.

Estilos: Podéis utilizar un archivo CSS propio o un CDN de TailwindCSS o BootstrapCSS para dar estilo a la página.

# Criterios de evaluación
La evaluación se realizará en función de los siguientes criterios:

Cumple con todas las funcionalidades del vídeo.
Tiene la estructura correcta de objetos con herencia.
El código está correctamente estructurado en los diferentes archivos indicados.
La página tiene una apariencia similar a la del vídeo y está bien presentada.
Se hace un uso correcto de CSS o del framework seleccionado.

# Entrega
La tarea debe entregarse en formato ZIP.
